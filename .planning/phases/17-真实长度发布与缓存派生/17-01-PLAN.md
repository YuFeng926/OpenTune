</objective>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| frozen render boundaries -> RenderCache publish | 发布边界必须继续绑定真实 sample span |
| base cache -> resampled cache | resampled 结果只能从裁后真实音频派生 |
| sample truth -> logs / export / read windows | time 值只能做投影，不能重新主导事实边界 |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-17-02 | T | `RenderCache::addResampledChunk` | mitigate | resampled 只附着到 revision 匹配的已发布 base chunk，并沿用同一组 sample span。 |
| T-17-03 | I | export / read / diagnostic time values | mitigate | 所有 time 值仅由 sample truth 投影得到，不允许反向成为 cache 发布事实。 |
</threat_model>
