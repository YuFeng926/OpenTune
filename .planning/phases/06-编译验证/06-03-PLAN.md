VST3 Debug build:
```bash
cmake -DFORMATS=VST3 -DCMAKE_BUILD_TYPE=Debug -B build-vst3
cmake --build build-vst3 --config Debug
```

VST3 Release build:
```bash
cmake --build build-vst3 --config Release
```

Expected output files:
- build-vst3/Debug/VST3/OpenTune.vst3/Contents/x86_64-win/OpenTune.vst3
- build-vst3/Release/VST3/OpenTune.vst3/Contents/x86_64-win/OpenTune.vst3

VST3 bundle structure (Windows):
```
OpenTune.vst3/
├── Contents/
│   ├── Resources/       # AI models, DLLs
│   │   ├── models/
│   │   │   ├── hifigan.onnx
│   │   │   └── rmvpe.onnx
│   │   ├── onnxruntime.dll
│   │   ├── DirectML.dll
│   │   └── D3D12/
│   │       ├── D3D12Core.dll
│   │       └── D3D12SDKLayers.dll
│   └── x86_64-win/
│       └── OpenTune.vst3  # Main plugin binary
```

From CMakeLists.txt:
```cmake
juce_add_plugin(OpenTune
    FORMATS Standalone VST3
    IS_ARA_EFFECT TRUE
)
```
