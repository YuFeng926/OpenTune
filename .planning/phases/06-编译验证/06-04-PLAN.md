  <files>build-standalone/Debug/, build-vst3/Debug/VST3/OpenTune.vst3/</files>
  <action>
    验证两个格式的运行时依赖是否一致：

    1. **DLL 版本一致性检查**：
       - 比较两个格式的 onnxruntime.dll 文件哈希
       - 比较两个格式的 DirectML.dll 文件哈希
       - 确保 DLL 来源相同

       ```powershell
       # 比较文件哈希
       $hash1 = Get-FileHash "build-standalone/Debug/onnxruntime.dll" -Algorithm MD5
       $hash2 = Get-FileHash "build-vst3/Debug/VST3/OpenTune.vst3/Contents/Resources/onnxruntime.dll" -Algorithm MD5
       if ($hash1.Hash -eq $hash2.Hash) {
         Write-Host "onnxruntime.dll 一致"
       } else {
         Write-Host "警告: onnxruntime.dll 不一致"
       }
       ```

    2. **模型文件一致性检查**：
       - 比较两个格式的 hifigan.onnx 文件哈希
       - 比较两个格式的 rmvpe.onnx 文件哈希

    3. **依赖完整性检查**：
       - 确认两个格式都包含所有必需的 DLL
       - 确认两个格式都包含所有模型文件

    如果发现不一致，记录到报告中。
